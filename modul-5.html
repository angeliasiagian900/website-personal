Cara Menampilkan Data dalam JavaScript

  1. Menggunakan innerHTML: Dengan innerHTML, Anda dapat menyisipkan HTML dan teks ke dalam elemen.
  2. Menggunakan textContent: Jika Anda hanya ingin menampilkan teks tanpa HTML, gunakan textContent.
  3. Menggunakan createElement dan appendChild: Untuk menambahkan elemen baru ke dalam halaman secara dinamis, Anda bisa membuat elemen baru dan menambahkannya ke dalam DOM.
